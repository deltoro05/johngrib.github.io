## Examples
```sh
$ # 실행하면 다음과 같은 내용이 출력된다
$ sw_vers
ProductName:    Mac OS X
ProductVersion: 10.15.2
BuildVersion:   19C57
```

```sh
 # productName 만 출력한다
sw_vers -productName

 # productVersion 만 출력한다
sw_vers -productVersion

 # buildVersion 만 출력한다
sw_vers -buildVersion
```
