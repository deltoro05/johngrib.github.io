## 범위

* `Integer.MAX_VALUE`는 $$2^{31} -1 = 2,147,483,647$$ 이다.
* `Integer.MIN_VALUE`는 $$-2^{31} = -2,147,483,648$$ 이다.

따라서 다음과 같은 컴파일 에러가 가능하다.

```java
/* Java */
// 정상
int a = -2147483648;
long b = -9223372036854775808L;

// compile error
int c = -(2147483648);
long d = -(9223372036854775808L);
```
