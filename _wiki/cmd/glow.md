## Examples

```sh
 # 설치한다
brew install glow

 # 도움말을 본다
glow --help

 # 마크다운 파일을 읽어 색칠해 보여준다
glow _wiki/glow-cmd.md

 # stdin으로 들어온 텍스트를 색칠해 보여준다
cat _wiki/glow-cmd.md | glow -

 # width 를 60으로 지정한다
glow _wiki/glow-cmd.md -w 60

 # dark / light 테마를 지정한다
glow _wiki/glow-cmd.md -s dark
```
