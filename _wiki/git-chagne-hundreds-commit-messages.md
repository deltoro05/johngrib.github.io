## 요약

다음 명령을 사용하면 `시작커밋` 부터 `마지막커밋` 범위의 커밋 메시지를 [[/cmd/sed]] 로 편집하게 된다.

```sh
git filter-branch --msg-filter \
 'sed -E "s/pattern/replace/"' \
 시작커밋..마지막커밋
```
