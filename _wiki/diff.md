## Examples
### 두 파일의 차이점 비교하기
```sh
$ diff FILE_A FILE_B
```

### 대소문자 무시
```sh
$ diff FILE_A FILE_B -i
```

### 공백문자 무시
```sh
$ diff FILE_A FILE_B -w
```

### 차이점을 좌우로 보여주기
```sh
$ diff FILE_A FILE_B -y
```

### 재귀 (디렉토리 비교)
```sh
$ diff DIR_A DIR_B -r
```

```sh
 # 차이가 있는 파일의 이름만 출력한다
$ diff DIR_A DIR_B -rq
```
