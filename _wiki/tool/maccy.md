## 개요

- macOS에서 사용하기 좋은 편리한 클립보드 히스토리 매니저.
- <https://maccy.app/ >
- <https://formulae.brew.sh/cask/maccy >

## 설치

```bash
brew install --cask maccy
```

설치가 완료된 다음, `시스템 설정` - `개인정보 보호 및 보안` - `손쉬운 사용` 에서 `Maccy`를 추가해준다.

## 설정

- 일반
    - 단축키: `command + F18`
    - 로그인 시 실행: `true`
    - 검색: `fuzzy`
    - 동작
        - 자동으로 붙여넣기: `true`
        - 서식없이 붙여넣기: `false`
- 고급
    - 어플리케이션 포커싱 방지: `true`
